The eBay SDK makes it easier to integrate an application, tool, or service into the eBay Platform. It is a set of libraries, sample applications, and documentation that can significantly cut time-to-market for new applications because it uses components developed and tested by eBay—ensuring reliability, quality, and stability.

To take full advantage of the eBay SDK, you need to retrieve a set of keys to interact with the eBay Platform. In order to retrieve these keys, you will need to join the eBay Developers Program.
